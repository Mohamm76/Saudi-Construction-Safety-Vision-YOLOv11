# Saudi-Construction-Safety-Vision-YOLOv11
"نظام ذكاء اصطناعي متطور يعتمد على خوارزمية YOLOv11 لمراقبة الامتثال بمعدات الوقاية الشخصية (PPE) في المواقع الإنشائية السعودية. يدعم المشروع معايير السلامة المهنية في المشاريع الكبرى (Giga-Projects) تماشياً مع مستهدفات رؤية 2030 لتعزيز بيئة عمل آمنة وذكية."
